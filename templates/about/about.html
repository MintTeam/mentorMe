About mentorMe

mentorMe is a JavaScript web application meant to facilitate the interaction between teacher and student.

It is a platform where a teacher can create tasks with title and description, add links to useful resources, set deadline. Teachers can assign the tasks they created to any registered student. Tasks can be editted or deleted before the deadline.

Students are expected to complete their tasks on or before the deadline. They can use the blog to ask questions about the tasks.
